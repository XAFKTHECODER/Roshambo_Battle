<h2>📝 About The Game</h2>
<p>
  <strong>Roshambo Battle</strong> brings the classic schoolyard hand-combat game straight into your developer console! Built entirely in Python, it features a bold ASCII art intro generated by <code>pyfiglet</code> and a clean, scoped game loop. 
</p>
<p>
  This project showcases Python basics, including random number generation, input validation, and function structure.
</p>

<hr />

<h2>⚡ Features</h2>
<ul>
  <li><strong>ASCII Art Intro:</strong> Uses <code>pyfiglet</code> for a retro arcade style terminal startup.</li>
  <li><strong>Emoji Support:</strong> Clean visual prompts using 🪨, 📄, and ✂️.</li>
  <li><strong>Robust Logic:</strong> Instantaneous result evaluation (Win/Loss/Tie).</li>
</ul>

<hr />

<h2>🕹️ How To Play</h2>
<ol>
  <li>Launch the game in your command line or terminal.</li>
  <li>When prompted, type your choice:
    <ul>
      <li><code>1</code> for Rock 🪨</li>
      <li><code>2</code> for Paper 📄</li>
      <li><code>3</code> for Scissors ✂️</li>
    </ul>
  </li>
  <li>The computer will instantly make its randomized choice.</li>
  <li>The winner is declared on screen with zero delay!</li>
</ol>

<hr />

<h2>🚀 Installation & Launch</h2>
<p>To run this game on your local computer, execute the following commands in your terminal:</p>

<pre><code># 1. Clone the repository
git clone https://github.com/XAFKTHECODER/Roshambo_Battle.git

# 2. Move into the project folder
cd Roshambo_Battle

# 3. Install the required ASCII text package
pip install pyfiglet

# 4. Start the game!
python main.py</code></pre>
